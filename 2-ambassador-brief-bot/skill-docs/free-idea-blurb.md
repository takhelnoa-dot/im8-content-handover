# Free Idea Blurb

When any deliverable is flagged as a "free idea" / "creator's choice" / "come up with it yourself", do NOT write prescribed content for that section. Instead, drop this blurb in the section so the creator knows they have latitude and where to draw inspiration from:

```
**This is a free idea — your pick.** Take any angle from your usual content that honestly connects to the focus area. Need a jumping-off point? Browse the IM8 idea bank here: [IM8 Idea Bank](https://drive.google.com/file/d/12gw4K3x-Zyv6TvgvsBtI7XH_BvQ-BHgn/view)

Let us know what you land on before you film so we can make sure it fits into the broader campaign.
```

## When to use this blurb

1. Creator request explicitly includes "free idea(s)", "creator's choice", "he/she can come up with one", or similar latitude-granting language
2. Any deliverable description that ends with "... and [N] free ideas"

## Where to place it

Inside the specific deliverable section it applies to. If the creator has:
- 4 Reels (2 prescribed focuses + 2 free ideas) → Reels 1-2 get prescribed content, Reels 3-4 each get the free-idea blurb
- 1 Reel + 1 Story Set where the story is "your choice" → the Reel gets prescribed content, the Story Set section gets the free-idea blurb

## Idea Bank URL (canonical reference)

`https://drive.google.com/file/d/12gw4K3x-Zyv6TvgvsBtI7XH_BvQ-BHgn/view`

Whenever the user requests ideas for a brief or asks for ideation support, link the idea bank in that section.
